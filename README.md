## ✨ **Features**

### 🔌 **Wallet Integration**
- Connect with MetaMask
- Auto-detect network (Polygon, BSC, Ethereum, Ganache)
- Switch networks with one click
- Display wallet balance and address

### 📊 **Dashboard**
- Real-time contract statistics
- User information and balances
- Network information display
- Quick actions panel

### 🎯 **Core Functionality**
- **Join Network**: Register new users with referral system
- **Binary Tree**: Visual representation of network structure
- **Hidden Wallets**: Create and manage hidden wallets (3 tiers)
- **Distribution System**: Automated profit distribution
- **Commission Claims**: Claim earned commissions
